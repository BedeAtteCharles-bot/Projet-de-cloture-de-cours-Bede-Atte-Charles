# Projet-de-cloture-de-cours-Bede-Atte-Charles
Application web de gestion de stock permettant de suivre les entrées, sorties et ruptures de produits. Le projet propose un tableau de bord clair et une interface simple destinée aux commerçants pour mieux gérer leur approvisionnement. Réalisé en HTML, CSS et JavaScript dans le cadre d’un projet de clôture de cours.
